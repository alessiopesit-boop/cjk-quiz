# Quiz CJK

Un quiz interattivo per imparare a distinguere i caratteri cinesi, giapponesi e coreani.

CJK sta per **C**hinese, **J**apanese, **K**orean: le tre grandi scritture dell'Asia orientale.

## Funzionalità

- Riconoscimento caratteri tra cinese, giapponese e coreano
- Attiva/disattiva le lingue che vuoi indovinare
- Link a Wiktionary per scoprire etimologia e significato
- Interfaccia disponibile in italiano e inglese
- Tracciamento punteggio con percentuale

## Come usarlo

Apri `cjk_quiz.html` in un browser. Nessuna dipendenza, nessun build.
